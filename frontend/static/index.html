How It Works

From empty floor plan to furnished interior — driven by a single structured prompt

Step 1 · Input Floor Plan

Upload any 2D architectural floor plan — even with text labels. The model preserves all walls, doors, windows and room shapes exactly.

Step 3 · Generated Interior

HomAIker adds furniture, removes text labels and applies the selected interior style — completely clean, top-down architectural output.

Step 2 · The Prompt

View example prompt used for this generation 
TASK
Take this floor plan and add furniture to it. REMOVE all text labels that appear in the input image (like room names). Keep the walls, doors, and room shapes exactly the same, but OUTPUT should have NO TEXT AT ALL - completely clean with only walls and furniture.

INPUT INSTRUCTIONS
Keep: walls, doors, windows, room shapes, layout
Remove: ALL text labels, room names, letters - the output must be text-free
Input image role: empty_floorplan_to_furnish
Number of outputs: 1

CRITICAL RULES
- DO NOT change the floor plan layout or room positions
- DO NOT add any new text labels - only keep labels that exist in the input image
- DO NOT add descriptive text like furniture names or descriptions
- KEEP the exact same walls, doors, windows, and room shapes
- ONLY add furniture inside existing rooms

GLOBAL CONSTRAINTS
- do not change architecture: True
- keep walls doors windows fixed: True
- keep room count exact: 5
- do not add or remove rooms: True
- do not add new text labels: True
- keep scale realistic: True
- no furniture overlap: True

DESIGN BRIEF
- household: single parent
- persona lifestyle: plant lover
- hobby tags: yoga, music
- personality social: balanced
- routine: cooking daily
- needs must implement: natural light, easy cleaning
- desires implement if feasible: cozy, warm
- budget level: mid
- storage intensity: low

STYLE PACK
- style id: 1
- interior style: Bohemian (Boho)
- palette description: ivory base with colorful textile accents (mustard, teal, rust)
- materials description: rattan, cane, and woven jute, linen and cotton textiles with patterned rugs, handmade ceramics and natural stone accents

OUTPUT
Name: 2D_topview_plan
Camera: strictly_top_down_orthographic
Render mode: flat_2D_architectural_diagram

NEGATIVE PROMPT
text, labels, letters, words, room names, front view furniture, 3D objects, shadows, perspective, isometric, grid, collage, multiple views
